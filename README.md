<h2 id="english">🇬🇧 English</h2>

I create music software through two distinct product lines:

- **DAWWW-CORE**, a music studio that runs in the browser;
- **UnicorSoundEngine**, a separate collection of native instruments and audio effects.

They belong to the same music-tech portfolio, but they are different products. DAWWW-CORE does not load native VST plugins inside the browser.

<p align="center">
  <img src="assets/diagrams/music-portfolio-map.svg" width="900" alt="DAWWW-CORE and UnicorSoundEngine portfolio map">
</p>

## DAWWW-CORE

DAWWW-CORE brings the main parts of a music-production studio into the browser:

- arranger and timeline;
- sequencer and piano roll;
- mixer;
- built-in instruments and sampler;
- audio effects;
- automation and modulation;
- audio and stem export;
- project recovery.

The studio is designed around a simple idea: a music project should be easy to save, reopen and continue.

<p align="center">
  <img src="assets/diagrams/dawww-core-workflow.svg" width="900" alt="DAWWW-CORE project workflow">
</p>

### Portable `.dw` projects

DAWWW-CORE uses the `.dw` project format to keep a session together. A project can carry its musical structure, settings and referenced material so the work can continue beyond a single browser session.

```text
create → save → reopen → edit → export → continue
```

### Desktop and Android

The main experience is built for desktop browsers. An Android version is also maintained through Capacitor for music-making and project access on compatible mobile devices.

[Discover DAWWW-CORE](docs/dawww-core.md) · [Music portfolio](docs/project-overview.md) · [FAQ](docs/faq.md)

## UnicorSoundEngine

UnicorSoundEngine is the native audio-plugin line. It brings together seven instrument families:

| Family | Musical direction |
| --- | --- |
| **Piano** | Piano and keyboard sounds |
| **Percussion** | Percussive instruments and acoustic hits |
| **Bass** | Bass instruments and low-frequency textures |
| **Guitar** | Guitar-inspired instruments and textures |
| **Drum** | Drum synthesis and rhythm-focused sounds |
| **Orchestral** | Orchestral and cinematic instruments |
| **Rare** | Unusual instruments and less common sound sources |

The line also includes audio effects, presets, manuals and listening examples. Each instrument or effect has its own sound identity and interface.

[Discover UnicorSoundEngine](docs/unicorsoundengine.md) · [Audio effects](docs/fx-line.md) · [Listening and presets](docs/audition-review.md)

## Two different audio paths

| DAWWW-CORE | UnicorSoundEngine |
| --- | --- |
| Music studio in the browser | Native plugin collection |
