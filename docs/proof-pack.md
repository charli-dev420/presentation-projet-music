# Public Proof Pack / Pack de preuves public

[FR](#francais) | [EN](#english)

## Francais

### Ce pack contient

| Preuve | Type | Niveau de confiance | Comment l'utiliser |
| --- | --- | --- | --- |
| Contrat `.dw` | Contrat produit | Fort comme definition, a rejouer comme release proof. | Comprendre la valeur de portabilite. |
| Desktop complete-gate | Synthese QA | Fort comme signal desktop. | Evaluer maturite desktop. |
| Android studio-grade | Gate actif attendu | A recertifier. | Evaluer readiness Android apres nouveau run. |
| Architecture UWdeVST | Documentation utilisateur | Fort comme coherence suite. | Comprendre synthés et UX plugin. |
| Suite FX | Carte produit | Moyen, tant qu'il n'y a pas de preuve publique par FX. | Comprendre le portefeuille sans surcharger. |
| Compliance/provenance | Politique source privee | A discuter sous cadre. | Evaluer publication, licences et distribution. |

### Ce pack ne contient pas

Pas de logs, binaries, APK, VST, presets prives, CSV, captures internes, device IDs, output folders ou secrets.

### Requete de preuve pour partenaire

Demander une demo ciblee: surface, scenario, critere de succes, artefacts rediges attendus, et ce qui peut rester sous NDA.

## English

The public proof pack contains written summaries only: `.dw` contract, desktop QA signal, active Android gate expectations, UWdeVST architecture, grouped FX positioning, and compliance discussion points. Raw artifacts remain private.
