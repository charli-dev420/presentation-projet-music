# User Flows / Flux utilisateur

[FR](#francais) | [EN](#english)

## Francais

### 1. Composer dans DAW Core

1. L'utilisateur cree un projet local.
2. Il choisit instruments, effets et arrangement.
3. Il sauvegarde dans un format projet portable.
4. Il recharge le projet sur desktop ou Android.
5. Il verifie que le rendu reste coherent apres import/export.

### 2. Utiliser le format `.dw`

1. Creer ou ouvrir un projet.
2. Ajouter instruments, effets, samples necessaires et snapshots.
3. Sauvegarder en `.dw`.
4. Importer sur une autre surface.
5. Verifier que checksum, assets embarques et snapshots restent valides.

### 3. Decouvrir Unicor SoundEngine

1. Lire la carte des plugins.
2. Identifier une famille: synthés ou FX.
3. Consulter le guide utilisateur public-safe.
4. Tester le role musical attendu.
5. Remonter un retour sur usage, ergonomie ou compatibilite.

### 4. Evaluer comme partenaire

1. Lire le one-pager.
2. Verifier les repos couverts.
3. Lire les preuves et limites.
4. Identifier les besoins: financement, QA, design sonore, packaging, distribution.
5. Demander une demonstration ciblee avec preuves adaptees.

## English

The main user journey is: create a local project in DAW Core, add instruments/effects, save it as a portable project, reload it across desktop or Android, and evaluate the result through public-safe proof summaries.

Partner evaluation starts with the one-pager, repository map, evidence page, limits, and the buyer brief before any private demo.
