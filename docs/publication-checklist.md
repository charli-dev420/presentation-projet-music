# Publication Checklist / Checklist publication

[EN](#english) | [FR](#francais)

## English

Before updating this showcase, check that:

- DAW Core is described as a browser-based **web local-first DAW**.
- Unicor SoundEngine is described as the separate Synthé / FX / VST line.
- Links point to existing files.
- Referenced images exist in `assets/`.
- Evidence wording matches the current validation level.
- No secrets, local paths, product binaries, raw logs, or unreleased packages were added.
- The text helps a user, tester, collaborator, funder, recruiter, or product reviewer understand what to do next.
