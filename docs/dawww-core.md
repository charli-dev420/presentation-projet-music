# DAWWW-CORE

**DAWWW-CORE** is the main music-tech project.

It is a local-first browser DAW built around Web Audio, project continuity, `.dw` sessions, export, desktop workflows, Android testing, and practical QA proof.

It is not a native VST host. DAWWW-CORE does not load VST plugins in the browser.

---

## Product idea

A music project should be easy to create, save, reopen, test, export, and continue.

The central question is not only whether the app can produce sound. The real question is whether the musical work survives the round trip:

```text
create → save → reopen → play → edit → export → continue
```

---

## What DAWWW-CORE represents

- Browser-based DAW direction.
- Local-first project handling.
- Portable `.dw` project format.
- Web Audio instruments and built-in effects.
- Export and recovery workflows.
- Desktop and Android validation paths.
- QA and release-readiness work.

---

## What it is not

- It is not a native desktop VST host.
- It is not a cloud-first music platform.
- It is not the same product as Unicorn Sound Engine.
- It does not rely on VST plugins to function in the browser.

---

## Why it matters

DAWWW-CORE is the project that best shows the product direction: practical music software, project continuity, audio behavior that can be tested, and a user workflow that can be explained without requiring access to private source code.
