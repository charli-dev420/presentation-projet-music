<h2 id="english">🇬🇧 English</h2>

## What is DAWWW-CORE?

DAWWW-CORE is a music-production studio that runs in the browser. It includes an arranger, sequencer, piano roll, mixer, instruments, sampler, effects, automation and export tools.

## What is the `.dw` format?

`.dw` is the DAWWW-CORE project format. It is used to keep the musical session together so it can be saved, reopened and continued later.

## Does DAWWW-CORE need an internet connection?

DAWWW-CORE is built with a local-first approach. Many parts of the studio are intended to work from the user’s device, although some features may still depend on the browser, installation method or connected services.

## Does DAWWW-CORE run on Android?

An Android version is maintained through Capacitor for compatible phones and tablets. The desktop browser remains the main experience.

## Can DAWWW-CORE load VST plugins?

No. DAWWW-CORE uses Web Audio instruments and effects inside the browser. Native VST products belong to UnicorSoundEngine.

## What is UnicorSoundEngine?

UnicorSoundEngine is a separate collection of native instruments and audio effects for compatible desktop DAWs.

## Which instruments are included in UnicorSoundEngine?

The collection is organized around seven families: Piano, Percussion, Bass, Guitar, Drum, Orchestral and Rare.

## Are the projects open source?

This repository presents the products and their documentation. It does not publish the complete private source code unless a specific file is explicitly released under a separate licence.

## How can I get in touch?

- DAWWW-CORE: `contact@dawww-core-local.com`
- UnicorSoundEngine and general contact: `unicornwhodev@gmail.com`

<h2 id="francais">🇫🇷 Français</h2>

## Qu’est-ce que DAWWW-CORE ?

DAWWW-CORE est un studio de production musicale qui fonctionne dans le navigateur. Il comprend un arrangeur, un séquenceur, un piano roll, un mixer, des instruments, un sampler, des effets, de l’automation et des outils d’export.

## Qu’est-ce que le format `.dw` ?

`.dw` est le format projet de DAWWW-CORE. Il sert à garder la session musicale réunie afin de la sauvegarder, de la rouvrir et de la poursuivre plus tard.

## DAWWW-CORE a-t-il besoin d’une connexion Internet ?

DAWWW-CORE est construit avec une approche local-first. De nombreuses parties du studio sont pensées pour fonctionner depuis l’appareil de l’utilisateur, même si certaines fonctions peuvent encore dépendre du navigateur, du mode d’installation ou de services connectés.

## DAWWW-CORE fonctionne-t-il sur Android ?

Une version Android est maintenue via Capacitor pour les téléphones et tablettes compatibles. Le navigateur desktop reste l’expérience principale.

## DAWWW-CORE peut-il charger des plugins VST ?

Non. DAWWW-CORE utilise des instruments et des effets Web Audio dans le navigateur. Les produits VST natifs appartiennent à UnicorSoundEngine.

## Qu’est-ce que UnicorSoundEngine ?

UnicorSoundEngine est une collection séparée d’instruments et d’effets audio natifs pour les DAW desktop compatibles.

## Quels instruments composent UnicorSoundEngine ?

La collection est organisée autour de sept familles : Piano, Percussion, Bass, Guitar, Drum, Orchestral et Rare.

## Les projets sont-ils open source ?

Ce dépôt présente les produits et leur documentation. Il ne publie pas l’ensemble du code source privé, sauf lorsqu’un fichier précis est explicitement proposé sous une licence distincte.

## Comment prendre contact ?

- UnicorSoundEngine et contact général : `unicornwhodev@gmail.com`
