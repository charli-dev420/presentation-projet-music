<h2 id="english">🇬🇧 English</h2>

## What is this repository?

A public presentation of two music-tech product lines: DAWWW-CORE and UnicorSoundEngine.

## Is the source code public?

No. The repository presents the projects without publishing the private source code or sensitive material.

## What is DAWWW-CORE?

A local-first browser DAW built with Web Audio and portable `.dw` projects.

## Does DAWWW-CORE host native VST plugins?

No. Its instruments and effects use the browser / Web Audio path.

## What is UnicorSoundEngine?

A separate native plugin line for instruments, effects, presets, manuals and product distribution.

## What is `.dw`?

The portable DAWWW-CORE project format used for save, reopen, recovery and project continuity.

## What is the Android status?

Android is an active test target through Capacitor. Public claims should be tied to dated device validation.

## What can be reviewed publicly?

Product positioning, selected screenshots, short demos, audio examples, documentation and concise validation notes.

## What remains private?

Source code, credentials, signing material, unreleased binaries, private account information and sensitive technical details.

## Who is the repository for?

Technical and product reviewers, music-software collaborators, testers and people evaluating the project portfolio.

<h2 id="francais">🇫🇷 Français</h2>

## Qu’est-ce que ce dépôt ?

Une présentation publique de deux lignes music-tech : DAWWW-CORE et UnicorSoundEngine.

## Le code source est-il public ?

Non. Le dépôt présente les projets sans publier les sources privées ni la matière sensible.

## Qu’est-ce que DAWWW-CORE ?

Un DAW navigateur local-first basé sur Web Audio et les projets portables `.dw`.

## DAWWW-CORE charge-t-il des VST natifs ?

Non. Ses instruments et effets utilisent le chemin navigateur / Web Audio.

## Qu’est-ce que UnicorSoundEngine ?

Une ligne séparée de plugins natifs pour instruments, effets, presets, manuels et distribution.

## Qu’est-ce que `.dw` ?

Le format projet portable de DAWWW-CORE utilisé pour la sauvegarde, la réouverture, la récupération et la continuité projet.

## Quel est l’état Android ?

Android est une cible de test active via Capacitor. Les affirmations publiques doivent être liées à des validations datées sur appareils.

## Qu’est-ce qui peut être revu publiquement ?

Le positionnement produit, des captures sélectionnées, de courtes démonstrations, des exemples audio, la documentation et des notes de validation concises.

## Qu’est-ce qui reste privé ?

Les sources, credentials, éléments de signature, binaires non publiés, informations de comptes privés et détails techniques sensibles.

## À qui s’adresse le dépôt ?

Aux reviewers techniques et produit, collaborateurs music-software, testeurs et personnes évaluant le portfolio.
