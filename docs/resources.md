### Glossaire

| Terme | Definition |
| --- | --- |
| DAW | Station de travail audio numerique. |
| `.dw` | Format projet portable de DAW Core. |
| Snapshot | Etat sauvegarde d'un instrument ou effet. |
| VST3 | Format de plugin audio. |
| Local-first | Donnees et operations prioritaires en local. |
| QA audio | Validation fonctionnelle et sonore d'un workflow audio. |

### Pour un utilisateur

Commencer par DAW Core, puis comprendre le format projet portable, puis lire les familles synthés/FX. L'objectif est d'apprendre le systeme avant de demander un acces a des builds ou demos privees.

### Pour un acheteur ou collaborateur

Lire les preuves, limites, roadmap et besoins. Demander ensuite une demonstration ciblee avec un perimetre clair: desktop, Android, VST, synthés, FX, QA ou distribution.
