# Tutorials / Tutoriels

[FR](#francais) | [EN](#english)

## Francais

Ces tutoriels sont volontairement sans code. Ils expliquent quoi regarder et comment progresser dans l'ecosysteme.

### Tutoriel 1 - Comprendre DAW Core en 10 minutes

1. Lire [Project Map](project-map.md).
2. Identifier le role majeur de DAW Core.
3. Lire le flux "composer dans DAW Core".
4. Lire [Evidence](evidence.md) pour comprendre ce qui est deja prouve.
5. Lire [Roadmap](roadmap.md) pour voir ce qui reste a stabiliser.

### Tutoriel 2 - Comprendre un projet `.dw`

Un projet `.dw` est presente comme un conteneur portable: checksum, assets sampler necessaires, snapshots instruments/effets, recharge desktop/Android et re-export apres import. L'utilisateur doit surtout comprendre que le format vise la continuite du projet, pas seulement un export audio.

### Tutoriel 3 - Lire la suite synthés/FX sans se perdre

Les synthés et effets sont regroupes. La lecture recommandee est:

1. Choisir un besoin musical.
2. Identifier si le besoin releve d'un instrument ou d'un traitement FX.
3. Lire la description de famille.
4. Evaluer l'integration avec DAW Core.
5. Remonter un retour utilisateur exploitable.

### Tutoriel 4 - Preparer un retour testeur

Un retour utile contient: surface testee, type de projet, resultat attendu, resultat observe, reproductibilite, impact musical, et niveau de blocage. Il ne doit pas contenir de secret, chemin local ou fichier prive.

## English

The tutorial path is: understand DAW Core, understand the portable `.dw` idea, read the synth/FX suite as grouped product families, then prepare useful tester feedback. No source code or private artifacts are required to follow these tutorials.
