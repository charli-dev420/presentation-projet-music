## Francais

**DAW Core** est le projet majeur. **Unicor SoundEngine** regroupe synthés, effets, site VST et audition autour de ce coeur.

| Point | Synthese |
| --- | --- |
| Utilisateur | Musicien, producteur, testeur audio, partenaire audio software. |
| Probleme | Composer, sauvegarder, tester et transporter un projet musical sans perdre son etat. |
| Solution | Station audio locale-first + format `.dw` + validation desktop/Android + suite synthés/FX. |
| Preuves | Contrat de portabilite, QA active, docs synthés, mapping multi-repos. |
| Besoins | Stabilisation, design sonore, QA, packaging, distribution, partenariats. |

## English

**DAW Core** is the major project. **Unicor SoundEngine** groups synths, effects, the VST site, and audition surfaces around it. The product goal is a local-first audio workstation with portable projects and public-safe evidence of real validation work.
