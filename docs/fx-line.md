# FX Line

The FX line is the effects branch of Unicorn Sound Engine.

It is separate from DAWWW-CORE. These are native plugin projects and product-family materials, not browser-loaded VST modules inside DAWWW-CORE.

---

## Families

The current effect families are organized around practical audio treatment roles:

- analyzer;
- creative;
- delay;
- distortion;
- doubler;
- dynamics;
- EQ / filter;
- modulation;
- pitch / time;
- reverb;
- stereo.

---

## Product goal

The goal is not just to have isolated effects. The useful goal is a coherent line:

```text
clear role → tested DSP behavior → usable UI → preset/readme → package → distribution
```

Each effect should eventually be easy to understand from three angles:

1. **musical role** — what problem it solves;
2. **technical status** — what is implemented and tested;
3. **release state** — what remains before packaging or distribution.

---

## Current priority

The priority for the FX line is presentation and proof:

- README per effect repo;
- screenshots or UI captures;
- short audio before/after examples;
- build/package notes;
- known limitations;
- compatibility notes with DAWs.

## Français

La ligne FX est la famille d’effets d’Unicorn Sound Engine.

Elle doit être lue comme une ligne plugin séparée : analyse, delay, distortion, dynamics, EQ, modulation, pitch/time, reverb, stereo, creative et doubler.

La priorité n’est pas d’ajouter des effets au hasard. La priorité est de rendre chaque effet lisible, testable, documenté et distribuable.
