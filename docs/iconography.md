# Iconography / Iconographie

[FR](#francais) | [EN](#english)

## Francais

| Symbole | Sens | Usage |
| --- | --- | --- |
| Waveform | Audio, rendu, validation sonore. | DAW Core, QA audio, audition. |
| Keyboard | Instrument, synthé, MIDI. | Suite UWdeVST. |
| Sliders | Effets, mixage, macros. | Suite FX, controle utilisateur. |
| File check | Projet `.dw`, portabilite, checksum. | Sauvegarde, import/export. |
| Phone | Surface Android. | Validation mobile. |
| Shield | Public-safe, perimetre, preuves redigees. | Docs de publication. |

### Regles d'usage

- Une icône doit representer une action ou une surface claire.
- Ne pas multiplier les icônes pour decorer.
- Garder les familles synthés et FX lisibles comme groupes.

## English

Icons should identify real product surfaces: waveform for audio validation, keyboard for instruments, sliders for FX, file-check for `.dw`, phone for Android, and shield for public-safe documentation.
