<h2 id="english">🇬🇧 English</h2>

This repository presents one music-tech portfolio through two product lines that solve different problems.

| Product line | Main role |
| --- | --- |
| **DAWWW-CORE** | Browser DAW focused on project continuity, editing, playback, export and recovery. |
| **UnicorSoundEngine** | Native plugin line for instruments, effects, presets, manuals and distribution. |

The separation is intentional. DAWWW-CORE uses Web Audio inside the browser and does not host native VST plugins. UnicorSoundEngine covers the native plugin path.

## Shared direction

Both lines are developed around a practical review model:

```text
product surface → observable result → review → correction → release decision
```

For DAWWW-CORE, the observable result is a project that can be created, saved, reopened, edited and exported. For UnicorSoundEngine, it is an instrument, effect or preset that can be heard, compared, documented and accepted or revised.

## Public maturity

| Area | Public framing |
| --- | --- |
| **DAWWW-CORE desktop** | Implemented and under stabilisation / release preparation. |
| **`.dw` format** | Implemented as the portable project format. |
| **Android** | Active test target through Capacitor. |
| **UnicorSoundEngine** | Separate product family with instruments, FX and product-support material. |
| **Public proof** | Screenshots, short demos and dated summaries should support the strongest claims. |

<h2 id="francais">🇫🇷 Français</h2>

Ce dépôt présente un portfolio music-tech composé de deux lignes produit qui répondent à des besoins différents.

| Ligne produit | Rôle principal |
| --- | --- |
| **DAWWW-CORE** | DAW navigateur centré sur la continuité projet, l’édition, la lecture, l’export et la récupération. |

La séparation est volontaire. DAWWW-CORE utilise Web Audio dans le navigateur et ne charge pas de VST natifs. UnicorSoundEngine couvre la partie plugins natifs.

## Direction commune

Les deux lignes suivent une logique de revue concrète :

```text
surface produit → résultat observable → revue → correction → décision de release
```

Pour DAWWW-CORE, le résultat observable est un projet qui peut être créé, sauvegardé, rouvert, modifié et exporté. Pour UnicorSoundEngine, c’est un instrument, un effet ou un preset qui peut être écouté, comparé, documenté puis accepté ou corrigé.
