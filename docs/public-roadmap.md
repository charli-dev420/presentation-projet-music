<h2 id="english">🇬🇧 English</h2>

This roadmap focuses on visible public improvements.

## P1 — Product visibility

- add a complete DAWWW-CORE workspace screenshot;
- add an arranger or mixer view;
- add a `.dw` save / reopen demonstration;
- add one export example;
- add a UnicorSoundEngine product mosaic.

## P2 — Listening and platform proof

- publish short instrument examples;
- publish one or two FX before / after examples;
- add a dated Android validation snapshot;
- add one plugin interface and one FX interface.

## P3 — Presentation polish

- add a preview of the one-pager or deck;
- update the visual index;
- remove obsolete duplicate documents;
- keep the README focused on the shortest useful path.

This roadmap is a presentation plan, not a release-date promise.

<h2 id="francais">🇫🇷 Français</h2>

Cette roadmap se concentre sur les améliorations publiques visibles.

## P1 — Visibilité produit

- ajouter une capture complète du workspace DAWWW-CORE ;
- ajouter une vue arrangeur ou mixer ;
- ajouter une démonstration sauvegarde / réouverture `.dw` ;
- ajouter un exemple d’export ;
- ajouter une mosaïque produit UnicorSoundEngine.

## P2 — Écoute et preuve plateforme

- publier de courts exemples d’instruments ;
- publier un ou deux exemples FX avant / après ;
- ajouter une validation Android datée ;
- ajouter une interface de plugin et une interface FX.

## P3 — Finition de la présentation

- ajouter un aperçu du one-pager ou du deck ;
- mettre à jour l’index visuel ;
- supprimer les documents obsolètes ou redondants ;
- garder le README centré sur le parcours utile le plus court.

Cette roadmap est un plan de présentation, pas une promesse de date de sortie.
