---

<h2 id="english">🇬🇧 English</h2>

Audio products cannot be evaluated through code alone. Instruments, effects and presets also need structured listening and comparison.

## Review targets

| Target | What to review |
| --- | --- |
| **Instrument presets** | Musical role, usability, naming and sound quality. |
| **FX presets** | Effect behavior, control and usefulness. |
| **Audio examples** | One notes, chords, loops and before / after renders. |
| **Manuals** | Product explanation and clarity. |
| **Product pages** | Alignment between the public description and the real product. |

## Review states

```text
accepted → needs revision → rejected → excluded from release
```

These states keep release decisions explicit and prevent every generated or modified asset from being treated as equally usable.

## Useful outputs

- listening notes;
- accepted and rejected lists;
- correction priorities;
- release candidates;
- structured CSV or JSON exports;
- short summaries for testers and collaborators.

<h2 id="francais">🇫🇷 Français</h2>

Un produit audio ne peut pas être évalué uniquement par le code. Les instruments, effets et presets demandent aussi une écoute et une comparaison structurées.

## Cibles de revue

| Cible | Ce qu’il faut revoir |
| --- | --- |
| **Presets d’instruments** | Rôle musical, facilité d’utilisation, nommage et qualité sonore. |
| **Presets FX** | Comportement de l’effet, contrôle et utilité. |
| **Exemples audio** | Notes seules, accords, loops et rendus avant / après. |
| **Manuels** | Explication et clarté produit. |
| **Pages produit** | Cohérence entre la description publique et le produit réel. |

## États de revue

```text
accepté → à corriger → refusé → exclu de la release
```

Ces états rendent les décisions de release explicites et évitent de traiter chaque asset généré ou modifié comme également utilisable.

## Sorties utiles

- notes d’écoute ;
- listes acceptées et refusées ;
- priorités de correction ;
- candidats de release ;
- exports structurés CSV ou JSON ;
- synthèses courtes pour testeurs et collaborateurs.
