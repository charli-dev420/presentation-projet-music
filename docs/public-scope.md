# Publication Scope / Perimetre de publication

[EN](#english) | [FR](#francais)

## English

### What Belongs Here

This repository is a public presentation surface for the music portfolio. It can include product framing, diagrams, banners, visual assets, user flows, tutorials, evidence summaries, QA vocabulary, current status, resources, and collaboration needs.

### What Stays Out

The repo should not include source code from the represented products, product binaries, unreleased packages, secrets, local machine paths, user sessions, raw logs, or material that cannot be shared cleanly.

This boundary should stay short and practical. The rest of the repository should focus on explaining the products.

## Francais

### Ce Qui A Sa Place Ici

Ce repo est une surface publique de presentation pour le portfolio musique. Il peut inclure cadrage produit, diagrammes, bannieres, assets visuels, flux utilisateur, tutoriels, syntheses de preuve, vocabulaire QA, statut courant, ressources et besoins de collaboration.

### Ce Qui Reste Hors Repo

Le repo ne doit pas inclure code source des produits representes, binaires produit, packages non publies, secrets, chemins machine locaux, sessions utilisateur, logs bruts ou materiel qui ne peut pas etre partage proprement.

Cette limite doit rester courte et pratique. Le reste du repository doit se concentrer sur l'explication des produits.
