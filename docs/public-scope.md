# Public Scope / Perimetre public

[EN](#english) | [FR](#francais)

## English

This repository is a public-safe presentation surface for the DAW Core music ecosystem.

### Included

- Product positioning and ecosystem map.
- Public-safe technical framing.
- User flows, tutorials, and reading paths.
- Partnership, funding, buyer, and recruiter material.
- High-level roadmap and readiness notes.
- Summarized QA and proof signals.
- Public or redacted diagrams and presentation assets.

### Excluded

- Source code and private repository content.
- VST binaries, APKs, installers, builds, release archives, stems, samples, private presets, and audio files.
- Secrets, server configuration, payment configuration, storage keys, admin details, account data, and endpoints.
- Internal QA folders, raw logs, CSV benchmark files, JSON dumps, device IDs, local paths, private screenshots, user sessions, and generated release artifacts.
- Synth, FX, audition, or distribution material that has not been rewritten as a public-safe summary.

### Review Rule

Every future file added here should answer one question: can a public partner read this without exposing implementation details, private infrastructure, or product-critical material?

## Francais

Ce repository est une surface de présentation public-safe pour l'écosystème musical DAW Core.

### Inclus

- Positionnement produit et carte écosystème.
- Cadrage technique public-safe.
- Flux utilisateurs, tutoriels et parcours de lecture.
- Matériel partenariat, financement, acheteur et recruteur.
- Roadmap haut niveau et notes readiness.
- Signaux QA et preuves synthétisées.
- Diagrammes et assets publics ou redacted.

### Exclus

- Code source et contenu de repositories privés.
- Binaires VST, APKs, installateurs, builds, archives release, stems, samples, presets privés et fichiers audio.
- Secrets, configuration serveur, configuration paiement, clés storage, détails admin, comptes et endpoints.
- Dossiers QA internes, logs bruts, CSV benchmark, dumps JSON, device IDs, chemins locaux, captures privées, sessions utilisateur et artefacts release générés.
- Matériel synthé, FX, audition ou distribution non réécrit comme synthèse public-safe.

### Règle de review

Chaque futur fichier ajouté ici doit répondre à une question: un partenaire public peut-il le lire sans exposer implémentation, infrastructure privée ou matériel critique produit?
