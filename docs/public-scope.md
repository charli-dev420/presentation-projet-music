# Public Scope / Perimetre public

This repository is a public-safe presentation surface.

## Included

- Product positioning.
- Public-safe technical framing.
- Partnership and funding needs.
- High-level roadmap.
- Non-sensitive diagrams.

## Excluded

- Source code.
- Private repositories.
- VST binaries, APKs, installers, builds, stems, samples, presets, or release archives.
- Secrets, server configuration, payment configuration, storage keys, admin details, or account data.
- Internal QA evidence, device IDs, local paths, logs, screenshots, or generated artifacts.

## Review rule

Every future file added here must answer one question: can this be read by a public partner without exposing implementation, private infrastructure, or product-critical material?
