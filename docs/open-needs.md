# Open Needs

This page lists the useful ways to help, test or collaborate around the music-tech projects.

---

## DAWWW-CORE

Useful feedback:

- browser DAW workflow review;
- Web Audio behavior and export checks;
- `.dw` project continuity feedback;
- save / reopen / recover / export scenarios;
- Android beta testing on real devices;
- UI/UX friction during real music sessions.

The best feedback is practical: what you tried, what happened, what felt unclear, and whether the musical project survived the workflow.

---

## Unicorn Sound Engine

Useful feedback:

- listening notes for synths and FX;
- preset role and naming feedback;
- manual clarity review;
- plugin packaging review;
- store/download flow review;
- compatibility notes with DAWs;
- product positioning and pricing feedback.

---

## FX Line

Useful feedback:

- which FX should be prioritized first;
- which effects need better UI or presets;
- before/after audio examples;
- build and installer validation;
- DAW compatibility checks;
- documentation review.

## Français

Les retours utiles sont concrets : tester DAWWW-CORE comme DAW web, vérifier la continuité `.dw`, écouter les synthés et FX, relire les manuels, tester les téléchargements, valider le packaging et aider à mieux positionner les projets.

Le meilleur retour décrit une action réelle, le résultat obtenu, ce qui bloque, ce qui manque et ce qui mérite d’être priorisé.
